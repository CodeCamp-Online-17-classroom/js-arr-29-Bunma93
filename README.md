### Lab114 Array: การลบค่าelement
จงเขียนฟังก์ชัน deleteTask(id) เพื่อลบ element ใน tasks ที่มี property ชื่อ id เท่ากับ id 

```JavaScript
const tasks = [
  { id: 1, name: 'Fishing' },
  { id: 2, name: 'Shopping' },
  { id: 3, name: 'Swimming' }
];

deleteTask(2);
console.log(tasks);
// [
//   { id: 1, name: 'Fishing' },
//   { id: 3, name: 'Swimming' }
// ]
```
